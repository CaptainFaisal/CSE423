CSE423 - Computer Graphics

To draw a line, call the function drawLine on the console, passing the coordinates of the start and end points and the line drawing algorithm as parameters. The algorithm can be DDA or BLA.
Sample call: drawLine(0, 10, 25, 17, "DDA");
currently, the screen is 32x18 pixels, so the coordinates should be between 0 and 31 for x and 0 and 17 for y.
